# Localizador de Imóveis

Aplicação WebGIS para localização e análise estatística de imóveis utilizando ArcGIS Enterprise e ArcGIS Maps SDK for JavaScript.

## Funcionalidades previstas

- Autenticação via ArcGIS Enterprise
- Visualização de municípios brasileiros
- Visualização de imóveis
- Clusterização de pontos
- Filtro por município
- Estatísticas interativas
- Relatórios

## Tecnologias

- HTML5
- CSS3
- JavaScript
- ArcGIS Maps SDK for JavaScript
- ArcGIS Enterprise
